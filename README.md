# dcgan #

dcgan is an implementation Deep Convolutional Generative Adversarial Networks

https://arxiv.org/pdf/1511.06434.pdf

### Dependencies ###

- Tensorflow (>= 8.0)

### How to Use ###

Please edit settings.py


```
#!bash

python trainer.py
```

You can see samples from Generative Network in samples.
